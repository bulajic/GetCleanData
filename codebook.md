# Code Book

## Subject and Activity

These variables identify the unique subject/activity pair the variables relate to:

 - **subject**: the integer subject ID.
 - **activity_label**: the string activity name:
  - *Walking*
  - *Walking Upstairs*
  - *Walking Downstairs*
  - *Sitting*
  - *Standing*
  - *Laying*
